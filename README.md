# Randy Phalla's Portfolio site

## Getting Started

### Install

`yarn`

### Development

`yarn start` and `http://localhost:8000/`

### Building

`yarn build`

### Test Build Locally

`yarn serve`

### Debugging

## Caching problem

`yarn clean` or `gatsby clean`
