# Randy Phalla's Portfolio site

## Getting Started

### Install

install Node v16 and `yarn`

### Development

`yarn start` and `http://localhost:8000/`

### Building

`yarn build`

### Test Build Locally

`yarn serve`

### Debugging
 ## Caching problem

`yarn clean` or `gatsby clean`

### Update Packages

- yarn
`yarn upgrade`

- npm
`npm outdated` and `npm update`
